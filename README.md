# bitBalance
Self-hosted Bitcoin wallet balance tracker for XPUB, YPUB and ZPUB using Electrs.

bitBalance is a lightweight self-hosted Bitcoin wallet balance tracker.

It allows you to monitor the balances of multiple wallets using XPUB, YPUB or ZPUB without exposing your data to third-party services.

The app connects directly to your local Electrs server, ensuring full privacy and sovereignty.

Designed to run on Umbrel.
